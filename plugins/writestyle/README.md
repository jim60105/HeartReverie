# writestyle

寫作風格指令，引導 LLM 以第一人稱內省式敘事、日系輕小說風格進行創作，參考田中ロミオ的文風特徵。

## 運作原理

外掛注入兩段提示詞片段：

| 片段 | Priority | 說明 |
|------|----------|------|
| `writestyle` | 100 | 完整風格規格，涵蓋詞彙、句式、語調、節奏、感官描寫、對話、段落結構等維度 |
| `writestyle_reinforce` | 800 | 風格強化指令，覆寫先前文本風格，並要求減少省略號使用 |

### 風格核心特徵

- **人稱**：第一人稱（「我」），嚴格限定主角視角
- **內在聲音**：大量內心獨白、自問自答、碎碎念、自我評價
- **感官體驗**：主觀感官細節高頻出現，視覺／聽覺／觸覺為主
- **語言基調**：口語化為底，需要時切換至精確描寫詞彙
- **情緒表達**：透過思緒、語氣、微動作自然流露
- **節奏控制**：長短句交替——長句用於思考與描寫，短句用於反應與對話

### 參考作者

田中ロミオ（Tomoya Tanaka），代表作品：家族計畫、CROSS†CHANNEL、人類は衰退しました。
