# options

從 LLM 輸出中提取結構化行動選項區塊，在前端渲染為 2×2 按鈕面板，點擊可複製選項文字。

## 運作原理

外掛透過 `promptFragments` 注入提示詞指令，要求 LLM 在每次回覆末尾輸出 `<options>` 標籤，內含 4 個行動選項，格式為 `1:【選項文字】`，每個選項約 20 字。

前端模組在渲染階段提取 `<options>...</options>` 區塊，解析編號選項列，轉換為 CSS Grid 2×2 按鈕面板。點擊按鈕後將選項文字複製至剪貼簿並顯示「已複製!」回饋。

### 選項格式

LLM 輸出範例：

```
<options>
1:【前往圖書館查閱資料】
2:【直接詢問管理員】
3:【繞到後門偷偷觀察】
4:【先回家準備工具】
</options>
```

解析器支援多種格式變體：`1:【文字】`、`1: 文字`、`option1: 文字`。
