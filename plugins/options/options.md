    rules:
        - 選項應爲客觀的陳述句，不包括行動的後果，僅包括角色行爲、對話或內心想法
        - 不省略選項內容中的角色完整名稱，寫明做出選項內行爲的角色名
        - 單個選項的字數在 20 字左右，在序號後用"【】"包裹完整選項內容
        - 前文中的選項內容已利用正則進行隱藏，必須忽視前文的影響，重新輸出 20 字左右的選項
        - 用完整的"<行動選項>"中文 xml 標籤包裹整個模塊
        - 不要遺漏結束標籤"</options>"
        - 不要遺漏結束標籤"</options>"
        - 不要遺漏結束標籤"</options>"
    option內容示例：【“只是爲了搜查證據而已。” 維奧萊塔強作鎮定地解釋，同時暗中觀察對方】
    special: 在角色失去行動能力時，將第三、第四個選項改爲他人視角中角色的遭遇
    format:
'
<options>
1:【最合理的行動，繼續推進當前劇情】

</options>
'
