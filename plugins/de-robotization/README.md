# de-robotization

提供去機械化寫作指令，防止 LLM 在高智商角色對話中使用 AI 式的機械語氣與分析性思維模式。

## 運作原理

外掛透過 `promptFragments` 注入提示詞指令，列出禁用詞彙與禁止句型，並提供正確與錯誤的範例對照，引導 LLM 以人類直覺式思維取代數據分析式表達。

### 核心規則

- **禁用詞彙**：數據、演算法、機率、最優解、賀爾蒙水平等技術性詞彙
- **禁止句型**：「基於我的分析……」、「從生物學角度……」等分析性開頭
- **替代策略**：以口語化表達取代專業術語，以直覺、偏好、情感驅動決策
- **自檢機制**：每次生成前檢查是否混入技術術語或過於流暢的書面語
